FSharp.Formatting

  1. Берет fsx или md файлы.
  2. Творит магию.
  3. Выдает HTML / LaTeX файлы.
  4. Могет в подсветку синтаксиса и вывод типов для HTML.

  • Собственные обработчики деревьев.

Что было сделано

Приблизительно

  1. Выяснили, что жрет Reveal.Js.
  2. Научили FSharp.Formatting генерировать соответствующую структуру.
  3. Научили FSharp.Formatting обрабатывать конфиги.
  4. Научили Reveal.Js использовать FSharp.Formatting для анализа F#.
  5. Автоматизировали генерацию презентаций.
  6. Написали сервер для realtime представления.
  7. Собрали все в один проект для упрощенной работы.

function copyWithEvaluation(iElem, elem) {
    return function (obj) {
        var newObj = {};
        for (var p in obj) {
            var v = obj[p];
            if (typeof v === "function") {
                v = v(iElem, elem);
            }
            newObj[p] = v;
        }
        if (!newObj.exactTiming) {
            newObj.delay += exports._libraryDelay;
        }
        return newObj;
    };
}
